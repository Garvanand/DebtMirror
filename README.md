DebtMirror is the debt negotiation and borrower rights enforcement module for AgentOS. 
It levels the playing field against predatory recovery tactics by equipping borrowers with mathematical leverage, institutional asymmetry breaking, and exact legal defenses using RBI Master Circulars.

## 🌟 Core Features

- **Mathematical Leverage Engine**: Actively calculates the bank's hidden NPA provisioning burden (0% for SMA, 15% for sub-standard) to advise borrowers exactly when they have maximum negotiation leverage.
- **CIBIL Engine**: Breaks down the precise, long-term impact of choosing an OTS (One Time Settlement) over a standard closure, preventing borrowers from accidentally nuking their credit files without knowing the consequences.
- **Regulatory Truth-Detector**: An advanced ChromaDB RAG filled with RBI Master Circulars and Supreme Court judgements. If a bank tells a borrower they "cannot restructure an unsecured loan," the RAG intercepts the claim and provides the exact circular refuting it.
- **Legal Draft Generator**: Auto-generates heavily documented, formal Hardship Letters and OTS Counter-Proposals, aggressively injecting non-negotiable legal disclaimers on every page.

## 🚀 Quick Start (Development)

### 1. Build the Backend
```bash
# Clone the repository
git clone https://github.com/your-org/agentos.git
cd Day11_DebtMirror

# Copy the environment file
cp .env.example .env

# Spin up the FastAPI server, Celery worker
docker-compose up -d --build
```

### 2. Seed the Regulatory Corpus
Before using the RAG features, you must index the RBI Master Circulars into ChromaDB:
```bash
python scripts/seed_corpus.py
```

## 🔒 Critical Legal Disclaimer
**DebtMirror is a negotiation intelligence tool, NOT a legal advisor.**
Every output generated by this system must structurally include the hardcoded disclaimer:
> *"This is informational guidance based on publicly available RBI regulations. For complex debt matters, consult a qualified advocate or CIBIL-certified credit counselor."*
